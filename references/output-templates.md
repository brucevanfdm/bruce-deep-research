# 输出模板参考

以下模板供撰写报告时参考。根据实际调研内容灵活调整，不必逐字套用。

## 通用规范

所有报告遵循以下原则：

- **决策导向**：报告的目的是帮用户做决策，不是堆砌信息。每个章节都要回答"所以呢？"
- **结论前置**：开头放执行摘要，让决策者 30 秒内抓住核心结论
- **置信度标注**：关键判断标注信息可靠程度
  - 🟢 **高置信**：多个可靠来源交叉验证
  - 🟡 **中置信**：单一可靠来源，或多个非权威来源
  - 🔴 **低置信**：推测、间接推断、单一非权威来源
- **引用规范**：每个关键事实和数据用脚注引用本地素材文件，同时保留原始 URL
  ```
  该公司 2024 年完成 B 轮融资 5000 万美元[^1]。
  [^1]: [sources/03-crunchbase-funding.md](sources/03-crunchbase-funding.md)（原始来源：https://...）
  ```
